- 객체(object)
	- 일상 생활에 있는 대상을 코드로 표현
	- 변수와 값, 함수를 가지고 있음
	- 객체를 이용하면 이미 만들어진 데이터와 함수를 사용할 수 있음
- 객체 사용 방법
	1. 객체를 생성(객체에 따라서 이미 생성되있는 객체)
	2. 객체.변수 = 데이터,   객체.함수()
	. 을 이용해서 객체가 가지고 있는 함수 및 변수, 값을 사용
	. 연산자는 객체 안에 있는 .
	ex) document.write(); -> document 객체 안에 있는 write함수를 사용하겠다는 말
		document, history 객체들은 각자 역할이 있음.
